# Игрушечный пример: 2‑уровневая GRA (псевдокод)

Пример, как работает GRA Мета‑обнулёнка в самом простом случае двух уровней: локальный и мета. [file:1077]

---

## 1. Структура уровней

- Уровень 0 (локальный): несколько доменов \(a \in \{1,\dots,N\}\).  
  - Примеры: экономика, медицина, экология.  
- Уровень 1 (мета): согласование всех доменов между собой.  

Цели:

- \(G_0^{(a)}\) — локальная цель домена \(a\).  
- \(G_1\) — мета‑цель согласованности (без конфликта и без нарушения аланского закона).  

---

## 2. Псевдокод 2‑уровневой обнулёнки

```text
# Данные:
#   {Ψ_0[a]}     — состояния доменов на уровне 0
#   {G0[a]}      — локальные цели G_0^{(a)}
#   G1           — мета-цель уровня 1
#   ε0, ε1       — пороги пены для уровней 0 и 1
#   η0, η1       — шаги для градиентного шага по пене

# Функции:
#   Foam0(Ψ_0[a], G0[a])        — локальная пена Φ^(0) для домена a
#   Foam1(Ψ_0[1..N], G1)        — мета-пена Φ^(1) для набора доменов
#   Grad0(Ψ_0[a], G0[a])        — ∇_Ψ Φ^(0)
#   Grad1(Ψ_0[1..N], G1)        — ∇_Ψ Φ^(1) по всем Ψ_0[a]

Procedure GRA_2level_Obnulyonka():
    # 1. Локальное обнуление (уровень 0)
    for a in {1..N}:
        while Foam0(Ψ_0[a], G0[a]) > ε0:
            g0 = Grad0(Ψ_0[a], G0[a])
            Ψ_0[a] = Ψ_0[a] - η0 * g0

    # 2. Мета-обнуление (уровень 1)
    while Foam1(Ψ_0[1..N], G1) > ε1:
        g1_all = Grad1(Ψ_0[1..N], G1)
        for a in {1..N}:
            Ψ_0[a] = Ψ_0[a] - η1 * g1_all[a]

    return Ψ_0[1..N]
